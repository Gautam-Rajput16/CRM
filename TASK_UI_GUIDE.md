- Touch-friendly buttons

## Best Practices for UI Usage

### Creating Tasks
1. Use clear, action-oriented titles
2. Provide context in descriptions
3. Set realistic deadlines
4. Choose appropriate priority
5. Add relevant tags

### Viewing Tasks
1. Use filters to focus on specific tasks
2. Check overdue tasks first
3. Review pending tasks regularly
4. Monitor in-progress items
5. Archive completed tasks

### Updating Tasks
1. Update status as you work
2. Add notes for important updates
3. Communicate blockers
4. Complete tasks promptly
5. Keep information current
